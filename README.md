    1. telescope dependencies : [fd-find](https://github.com/sharkdp/fd?tab=readme-ov-file#installation).

&nbsp;
    3. clipboard dependencies : [wl-clipboard](https://github.com/bugaevc/wl-clipboard).
        -  `sudo apt install wl-clipboard`
